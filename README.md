MoodGallery – Trabalho EAD

A MoodGallery é uma aplicação web que funciona como uma rede de partilha temática e gestor de imagens, permitindo aos utilizadores publicar, organizar e explorar conteúdos visuais por categorias e estados de espírito.

O projeto foi desenvolvido no âmbito do Trabalho EAD, com foco em funcionalidades como upload de imagens, categorização, autenticação de utilizadores, notificações e estatísticas visuais.

Funcionalidades Principais

Upload e gestão de imagens

Organização por categorias temáticas

Sistema de utilizadores (login e registo)

Sistema de notificações

Pesquisa por categorias e descrição

Geração de gráficos estatísticos

Interface simples e funcional

 Bibliotecas Utilizadas
 Dependências Externas

Flask – Framework web principal

Werkzeug – Manipulação segura de ficheiros

Flask-Bcrypt – Encriptação de passwords

Pillow (PIL) – Processamento e edição de imagens

Matplotlib – Geração de gráficos e estatísticas

 Standard Library (Python)

os – Operações sobre o sistema de ficheiros

time – Gestão de tempo e delays

pathlib – Manipulação de caminhos de ficheiros

json – Leitura e escrita de dados

datetime – Gestão de datas e horas
